#### 一个使用jsdoc生成接口文档的简单但又完整的项目

1. jsdoc的配置文件：配置输入文件、命令行参数、模板主题、插件使用
2. jsdoc标签语法
3. 根据需要开发相关的插件，进行一些自定义的操作，不需要写在源代码的一些操作。

#### lib目录里面的插件
